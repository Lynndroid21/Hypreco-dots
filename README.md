# Hypreco-dots
A basic edit to the End4 dotfiles for anti-ai users.

I started this project due to the fact I personally have a great distaste for AI in general (more specifically how it's used and the damage it does to society) and many dotfiles I found (some OS like Nyarch as well) come with ai features, so I decided to make my own edits for selfhosting to replace the AI modules installed in said dotfiles.

Absolutely zero hate to any of these developers, their work is truly amazing! I just prefer OpenAI to have no part in my system, so I'm making this opinionated fork for people who agree! This does require you to have installed those og dotfiles though, as this is an edit of those and semi-heavily relies on scripts in that install.

# Installation
As stated above, you NEED to install End4's dotfiles first, as these configs and scripts are based around the filesystem he made. 

Dependencies (separate from the End4 deps installed ahead of time):

* Docker Desktop
* Self Hosted Search engine (I use SearXNG, but if there's another one that exists that you wanna use go ahead)
* Kate Advanced text Editor (you can find it on flathub, I highly recommend using it for code edits, it just simplifies your workflow nicely.)

Other used applications:

* Localsend
* Kitty Terminal (it works the best imo for theming)
* Brave (most browser callbacks in these files will call for it)
* Deezer Desktop (You can use spotify-- ew-- instead of deezer, I personally enjoy deezer more)
* Lazyvim (for those who enjoy terminal file browsing more)
* Dolphin (default file browser used)

Most of these dependencies are fairly simple to get running, the real work is inserting certain labels for your specific docker components. Luckily I'm working on making several scripts to automate certain containers to run on startup. This is currently a WIP, so check back in for updates on development!
